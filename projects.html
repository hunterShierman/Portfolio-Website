Ninja Side Scoller Game!

My friends and I created this at our first Hackathon. This project was a huge deal for me because it was my first experience programming and creating in a team setting. Together we created our production plan and divided up the work between the three of us. This game was created in python and using the pygame import. During the process we created the sprites, character animations, game enviroment, enemy AI, and detection hitboxes.
